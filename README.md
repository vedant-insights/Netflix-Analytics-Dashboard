# Netflix Analytics Dashboard

## Overview
This project is a Power BI dashboard analyzing Netflix content trends including:
- Total Movies & TV Shows
- Genre distribution
- Country-wise content
- Release trends over time

## Files Included
- Netflix_Dashboard_Report.pdf

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization

## Key Insights
- Most content is from USA & India
- Drama & International genres dominate
- Rapid growth in content after 2015
